BETA

Model Runner is a light-weight job management web application. The purpose is to expose models to be run as jobs in the cloud. If your model takes a set of input files and produces a set of output files, you can expose it via the web to a broader audience. Sharing tools is now much easier.

Why

Model Runner was built out of the need to easily share our development planning models with colleagues and partners throughout the world. These models are powerful, but require a lengthy installation process to be run on personal computers. Now, with Model Runner, anyone can user our tools without operation roadblocks. It makes it easy for us to collaborate with partners and at the same time allows us to open our tools to a greater audience.

How

Model Runner uses a simple primary/worker architecture to distribute model runs (jobs). Any model that can be run as a Linux process that takes input files and produces output files can be deployed and shared. Model Runner is powered by Python, Tornado, Redis and NGINX. It is open source (contributions encouraged) and licensed under the WTFPL.
